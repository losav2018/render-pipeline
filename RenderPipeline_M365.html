Connect to Microsoft 365
Enter your Azure App credentials to sync data automatically with an Excel file on SharePoint or OneDrive.
  1. Go to portal.azure.com → Azure Active Directory → App registrations → New registration
  2. Add redirect URI: https://your-hosted-url.com (or http://localhost for local testing)
  3. Under API permissions add Files.ReadWrite and Sites.ReadWrite.All
  4. Copy the Application (client) ID and your Tenant ID below
  5. Paste the SharePoint URL of the Excel file (or leave blank to use OneDrive root)
Found in Azure AD → Overview → Directory (tenant) ID
The file will be created automatically if it doesn't exist.
